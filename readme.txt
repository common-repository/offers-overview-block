== Description ==

Offers Overview Block for new WordPress Block Editor allow you to add offers overview to your website and customize it.

**Features**

* ready to use out of the box, no configuration needed,
* specify title, description, two additional contents, image and url for each offer,
* add as many offers as you need,
* display offers in a nice way, interesting for your website visitors,
* customize default styles settings using hooks programmatically (for theme authors),

== Installation ==

1. Install the plugin through the WordPress plugins screen directly or upload the plugin files to the "/wp-content/plugins/offers-overview-block" directory.
2. Activate the plugin through the "Plugins" screen in WordPress
3. Go to Block Editor and type /offers to use this block.
